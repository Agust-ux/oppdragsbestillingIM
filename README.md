# Oppdragsbestilling IM
En platform for forenklet oppdrags bestilling hos IM-Elvebakken. 

Platformen er for elever, lærere og eksterne kunder.

----
Elever
----
- Tar oppdrag
- Får oppdragene filtrert etter
    - MP
    - IT
    - VG1
    - VG2
    - Leveringsfrist
  - Innboks funksjon- varsel når de blir tildelt en oppdrag

----
Lærere - admin tilgang
----
- Assign oppdrag
- Ta opp oppdrag
- Alle oppdragene vises til ansvarlige lærere (Delt i MP og IT)
- Ser status på oppdragene de er ansvarlig for
- Se på elevers public profile- få info til å assigne oppgaver.

-----
Eksterne kunder
-----
- Innsikt i hva siden driver med
  - Tjenester vi tilbyr
- Kan sende forespørsel om en oppdrag
- Engangskode på første innlogging
- Kontakt
- FAQ
