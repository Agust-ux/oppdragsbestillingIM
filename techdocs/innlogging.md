# Innlogging 

----
Log-in
---
Elever og Lærere - Feide
Eksterne kunder - Logger inn med egen/ bedrift e-post
                - Får engangskode
                - Kommer seg inn og kan sende forespørsel

Om forespørsel er godkjent av en lærer
    - E-posten får tilgang for innlogging til varighet av oppdraget
    - Kan følge opp status av oppdraget og får også midlertidig chatt på nettsiden

Om forspørsel er **ikke** godkjent
    - Man får bedskjed på mail og kan svare om det er noe som ikke stemmer
    - Disse e-postene blir arkivert i systemet og slettes i 3 uker
