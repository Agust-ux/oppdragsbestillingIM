Oppdrags bestilling IM

Oppdragsbestilling IM Elvebakken spør om rettigheter til personvernet ditt. Dette dokumentet forklarer hva slags data vi bruker og hvordan denne infoen blir behandlet.

VIKTIG Å LESE NØYE

Hva slags data har vi om deg?

Når du bruker Oppdragsbestilling skal vi hovedsaklig ha nød til informasjon son følgende

  • Personlig data: Navn, profil bilde, bedrifts navn
  • Kontakt data: Telfon nummer, e-post
  • Oppdrags data: Tittel, beskrivelse, status, budsjett, varighet av oppdraget
Data type Hvorfor trenger vi dette?
Personlig data Vi henter dette for å vise din profil på nettsiden når du lager din public profile. Dette gjør det lettere for deg som bruker av nettsiden til å ta kontakt med andre og se deres arbeid.
Kontakt data Vi bruker kontakt infoen til å kontakte deg og sende oppdateringer angående oppdraget du har bestilt. Dette brukes også av elever eller lærere som jobber med oppdraget for å ta kontakt med deg.
